# BMAアーキテクチャ：論理主権の技術フレームワーク

## 1. 概要
バイモーダル・メモリ・アーキテクチャ（BMA）は、『崩壊の日』で描かれる「論理主権」の技術的基盤です。AIを中央集権的なクラウド処理から、分散型のローカル論理抽出へと転換させます。

## 2. コアコンポーネント
- **LCS（論理崩壊ストレージ）**: 膨大なビデオ/データストリームを48バイトの論理不変量に圧縮します。
- **原子論理カード**: ローカル知識ベース（Obsidianなど）に保存される独立した思考ユニット。
- **血縁アンカリング**: AIの論理コアをユーザーの個人/家族データに紐付けるプロトコル。

## 3. 現実世界での実装
私たちのコードベースでは、`lingxi_zettel_engine.py` が以下の方法でBMAの予備的な論理を実装しています：
1. 生データから「論理原子」をスキャン。
2. コンテキストを不変のハッシュ値に崩壊。
3. 長期的な推論のためのゼッテルカステン形式のカードを生成。
