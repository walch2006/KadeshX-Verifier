# 《红楼梦》后四十回逻辑还原 · 第一百五回

### 第一百五回 凤姐病终恩怨尽 贾琏悔过泪无声

荣国府的深秋，冷得透骨。那风穿过曾经雕梁画栋的长廊，如今只带起一阵阵腐朽木料的苦涩味和经年未扫的积灰。长廊尽头的漆皮早已剥落，露出了灰白的、被虫蛀过的内里，像是一道道丑陋的伤疤，在冷风中瑟瑟发抖。在府里最偏僻、最阴冷的一角，那个曾经威风八面、连走路都带风的二奶奶王熙凤，如今正蜷缩在一张散发着浓重药味与陈年霉味的旧榻上。

窗外，几株残败的芭蕉在寒风中剧烈地摇曳，那枯干的叶子互相摩擦，发出的沙沙声，在这死寂的夜里听起来格外惊心动魄，倒像是无数个被她生前害死的冤魂在暗处窃窃私语，又像是在为这位曾经的权势女性合唱一曲凄凉的挽歌。偶尔有一声夜枭的啼鸣划破长空，凄厉而短促，更添了几分末世的荒凉。屋里的光线昏暗得令人窒息，只有一盏残灯在风中剧烈地摇曳，灯芯早已结了厚厚的黑垢，火苗忽明忽暗，将凤姐蜷缩的身影在剥落的墙壁上拉扯得扭曲、诡异，宛如一个正在消散的残梦。

凤姐的脸已经瘦得脱了形，原本丰润的脸颊凹陷下去，只剩下巴掌大的一块。那双曾经顾盼生辉、藏着无数杀伐果断心机的丹凤眼，此时已深深地陷了下去，眼眶周围是一圈浓重的、化不开的青黑，像是死亡早已在那儿扎下了根，正一点点吸食着她残存的精气。她急促而艰难地喘着气，每一次呼吸都像是要耗尽全身仅存的一点生机，胸脯剧烈地起伏着，发出一种类似于破损风箱抽动的嘶哑声，每一次震颤都牵动着她那已经衰败的肺腑，带来一阵阵撕心裂肺的隐痛。

“平儿……平儿……”她颤抖着伸出手，那手早已枯瘦如柴，指甲缝里甚至于于还残留着因常年忧急而抠弄手心的痕迹，那是指甲嵌入皮肉留下的、暗红色的旧痕。在这一刻，她不再是那个统领千军万马、弹指间定人世荣辱的荣府大管家，只是一个在死亡边缘挣扎、被无边孤独与恐惧彻底吞噬的弱女子。

平儿早已跪在榻边，哭得双眼红肿如桃，她紧紧握住凤姐那只冰冷得没有一丝温度的手，试图用自己的体温去唤回那一点点消逝的暖意。她的声音哽咽得几乎听不清，带着一种近乎哀求的颤抖：“奶奶，我在这儿，平儿一直守着您呢……您别想那些不开心的，大夫说了，只要您静心养着，总会好起来的。您有什么放不下的，您尽管吩咐，平儿便是拼了这命，也一定给您办到。”

凤姐此时的心绪，已不是省事的悔恨所能涵盖，而是一场关于生命意义的终极坍塌。在那种半梦半醒的幻觉中，她看见了尤二姐那张清丽却带着血痕的脸，正对着她凄然一笑；她看见了张华那双充满了仇恨的眼睛，在黑暗中死死盯着她；她还看见了铁槛寺里那些因为她的贪婪、因为那三千两银子而彻底破碎的家庭，在那儿哭天抢地。她曾以为，只要手中握住了那把象征着权力的沉甸甸钥匙，就能锁住富贵，锁住宠爱，甚至于于能锁住命运的喉咙。可如今，当那把钥匙被死亡的锈迹斑斑侵蚀、变得沉重而毫无意义时，她才惊恐地发现，自己守护了一辈子的，竟然只是一座金漆剥落、内里早已腐烂生蛆的废墟。这种从权欲巅峰跌落到虚无深渊的幻灭感，让她感到一种透骨的寒意，比窗外的寒风更让她战栗。她省悟到，自己生前所有的“精明”，那些所谓的“杀伐果断”，在这一刻都成了上苍对她最辛辣、最无情的嘲讽。她想要求救，却发现自己连声音都发不出来，只能在那无边的黑暗中沉沦。

“我这辈子……争强好胜……到头来……落得个……家破人亡……”凤姐断断续续地吐着字，每一个字都像是带着血，带着她一生不甘的余温，“平儿……我后悔啊……若有来生……我宁愿去那荒山野岭……做一个……连名姓都没有的……烧火的丫头……哪怕日日烟熏火燎……也不要再进这……吃人的……富贵坑……这里头的血……太冷了……”

就在这时，房门被轻轻推开，贾琏走了进来。他看起来比往日苍老了十岁，原本白皙的脸上布满了暗沉的斑点和沧桑的皱纹。他身上那件昂贵的貂裘早已不知在哪个当铺里换了酒钱，只穿着一件略显单薄、袖口已经起毛的旧棉袍。他看着榻上那个曾让他恨得咬牙切齿、也曾让他敬畏得不敢抬头、更曾让他痴迷过的女人，心中像是打翻了五味瓶，酸楚、愤怒、悔恨与怜悯在那儿翻江倒海。

他想起了初婚时凤姐的娇憨与妩媚，那时她的笑声像银铃一样清脆；他想起了大观园里她谈笑间樯橹灰飞烟密、将上下几百口人治理得服服帖帖的威严；他也想起了她在得知他偷娶尤二姐时的毒辣与疯狂，那种不顾一切、要将世界同归于尽的决绝。

贾琏在榻边默默坐下，没有说一句宽慰的话，也没有像往日那样出言讽刺。在死亡这面巨大的、冷酷的镜子面前，所有的虚伪、所有的修饰、所有的爱恨情仇都瞬间失去了意义，变得苍白无力。他的这种沉默，原是是他内心最剧烈的翻江倒海。他恨自己，恨自己的荒唐无能，恨自己只会在温柔乡里逃避，恨自己在家族倾覆时甚至于于无法给发妻买一帖真正对症的好药，只能眼睁睁看着她在这里枯萎。他也怜她，怜这个女人为了贾家操碎了心，为了维持那点虚幻的尊荣得罪了所有人，最后却落得个众叛亲离、连个送终的人都难找的下场。这种繁难的情感被他深深地“藏”在微颤的指尖和低垂的眼帘下，而凤姐那毫无遮掩、近乎原始的惨状，则是命运对这段荒诞、扭曲婚姻最彻底的“露”。两人之间的这种静默，是恩怨散去后的死寂，也是对往昔错失岁月的无声祭奠。

凤姐费力地转过头，看着贾琏。在那一瞬间，她涣散的瞳孔里似乎闪过了一丝微弱的光，仿佛又回到了那个初见的午后，阳光洒在王府的假山上，那个尚未被权欲侵蚀、满心只有情郎的王家大小姐。

“二爷……你总算……来了……你还……怪我吗？”凤姐的声音轻得几乎要消失在风里，带着一种临终前的、卑微的试探。

贾琏再也忍不住，泪水夺眶而出，那泪水划过他沧桑的脸庞，滴落在凤姐那只冰凉的小手上。他哽咽着点头，大手终于覆上了那只枯瘦的手，紧紧握住：“不怪了，凤儿，我早就不怪了。咱们都错了……往后的日子……咱们都不争了，都不抢了……只要你能活过来……”

凤姐的执念，曾是那本怎么也算不完的账簿，是那份不愿屈居人下、要将所有人踩在脚下的骄傲。直到此刻，当生命之火即将熄灭时，她才真正迎来了觉醒。她终于省悟，那所谓的“管家奶奶”的头衔，不过是皇权垂怜下的一抹浮云，是封建礼教套在她脖子上的一道金锁，是贾家这座大厦消散前的一块装饰。她越是用力挣扎，那锁就勒得越紧。当她决定彻底放手、不再去计较那些得失荣辱的那一刻，她才感受到了一种前所未有的、极其讽刺的轻盈。这种觉醒虽然来得太迟，却是她这辈子唯一一次真正的自由，虽然代价是她的生命。

王熙凤的死，标志着荣国府最后一点“管理秩序”的彻底消散。她那种以暴制暴、以权谋私、甚至于于带点江湖气的手段，虽然为人不齿，却是那个末世家族维持表象繁荣的唯一手段。她是那个腐朽制度培养出来的最得力的刽子手，也是最可悲的陪葬品。当皇权的巨轮碾过贾家时，她作为这个家族最尖锐、最不安分的那根刺，必然最先折断。她的离去，不仅是一个女人的香消玉殒，更是那个依靠私人权谋、裙带关系和冷酷威压维持的旧世界崩解的缩影。

随着最后一声轻微的、像是卸下了千斤重担般的叹息，凤姐的手软软地滑落，指尖触碰到了那冰冷的、象征着她一生劳碌却终归虚无的床沿。她的眼睛微微闭上，嘴角竟然带着一抹解脱的微笑，仿佛她已经看见了在那白茫茫的大地尽头，有一片不需要算计、不需要争斗、也不需要权力的乐土。

平儿发出一声凄厉而压抑的哀号，整个人瘫软在榻边。贾琏没有动，他只是静静地坐着，任由眼泪纵横交错地爬满脸庞，任由那盏残灯在他面前彻底熄灭。他转过头，看向窗外那漆黑一片、深不可测的夜空，忽觉这偌大的荣国府，竟然冷得像一座巨大的、空旷的坟墓，而他，则是那个被留下来守墓的人。

这一回，写尽了脂粉阵里的血雨腥风如何归于寂灭，也写尽了名利场上的精明算计如何化作一声叹息。王熙凤，这个红楼世界里最鲜活也最繁难的女性，终究是在这秋风落叶中，完成了她那充满罪恶与哀怜的宿命轮回。

正是：
机关算尽太聪明，反误了卿卿性命。
生前心血付东流，死后恩怨随风影。
金钗断处情难续，寒灯灭时梦已醒。
从此尘寰无此女，空留余韵说凤鸣。
