# 《红楼梦》后四十回逻辑还原 · 第九十三回

### 第九十三回 焚稿断痴情魂去 传噩耗宝玉心崩

清晨的潇湘馆，被一层厚厚的寒霜笼罩着。昨夜那场大雪虽已停了，但空气中那股子沁骨的冷意，却仿佛要将人的魂灵都冻结起来。紫鹃伏在黛玉的床边，整个人已经哭成了一个泪人。那床上的女子，面色如玉，神情安详，若不是那早已冰凉的双手和停止了起伏的胸膛，倒真像是陷入了一场长长的美梦。

“姑娘……你终于是去了。”紫鹃抽噎着，声音沙哑得几乎听不见，“去了好，去了便不用在这腌臜地方受气了，不用再还那还不完的泪债了。”

紫鹃此时的心境是极为繁难的。她恨，恨这府邸的无情，恨那些长辈的冷酷；她也悲，悲黛玉这一生的孤苦，悲那“木石前盟”终究化作了泡影。可在那恨与悲之外，竟然还有一丝淡淡的欣慰——她看着黛玉临走时那副决绝的神情，知道自家的姑娘终于是用死亡这种最惨烈的方式，完成了对自己灵魂的救赎。

而在那头，荣国府的喧嚣才刚刚开始。昨夜的婚宴留下了满地的残红，婆子们正忙着打扫，却听闻了潇湘馆传来的噩耗。一时间，窃窃私语声如同受惊的群蜂，在回廊间嗡嗡作响。

“听说了吗？林姑娘去了。”
“嘘，小声点！今儿个可是二爷新婚头一天，说这些没影的话，小心被太太揭了皮。”
“唉，也是个苦命人，偏生赶在这个时候……”

消息传到王夫人耳中时，她正坐在暖阁里，手里端着一盏燕窝粥。听闻死讯，她的手猛地抖了一下，那细腻的瓷盏在桌缘磕出一声脆响，溅出了几点温热的液体，落在她那绣着金线如意的衣襟上。

“怎么就……这个时候去了？”王夫人皱着眉，语气里并没有多少哀伤，反而透着一股子被人搅了局的厌烦。在她看来，黛玉的死不仅是一桩丧事，更是对这场精心筹划的“冲喜”婚礼的一种诅咒。她担心的不是黛玉的命，而是这丧事会不会冲撞了宝玉的运势，会不会让外头那些查抄的官员觉得贾家已经气数将尽。

一旁的凤姐儿见状，忙拿了帕子替王夫人揩拭，一边拿眼角瞟着外头的动静，压低了嗓子道：“太太宽心，林妹妹那身子骨原是撑不住的，早晚的事儿。只是今儿个这日子……确实不凑巧。依我看，这事儿得压着，断不能让二爷知道。那潇湘馆离怡红院远着呢，咱们只管把门关紧了，说是林妹妹病重谢客就是了。”

王夫人点了点头，叹了口气：“也只能如此了。你去吩咐紫鹃那丫头，让她把嘴闭严实了。若走漏了风声，我决不轻饶。再者，那边的丧事，一切从简，莫要闹出大动静来。如今咱们家是什么光景，你也知道，能省一分是一分吧。”

凤姐儿应了一声，心里却是一阵阵的发虚。她想起黛玉临终前那焚稿的决绝，想起那吐在白绢上的点点红痕，只觉脊背一阵阵发凉。这大观园里的繁华，终究是透着一股子坟墓里的腐朽气。

这就是此时荣国府的真实面貌——在礼法的严苛与现实的残酷双重挤压下，人的情感已经被剥落得只剩下最冷硬的利益算计。林黛玉的生命，在这个庞大的、正在消散的家族机器面前，微薄得如同一片飘落的残叶，甚至于于不如一场法事的开销来得重要。每个人都在为自己的生存与体面挣扎，而那份最纯粹的真情，早已成了这废墟上最奢侈的祭品。

怡红院的新房里，红烛摇曳，残泪满台。宝玉正从昏沉的睡梦中惊醒。他猛地坐起身，看着周围那些大红的陈设，一时间竟分不清今夕何夕。宝钗正坐在镜前理妆，见他醒了，忙起身过来。

“二爷，可是魇着了？”宝钗声音温婉，透着一股子新妇的柔顺。她那双如水的大眼里，此刻藏着一抹不易察觉的忧虑。

宝玉没有理她，他的目光死死地盯着窗外。他觉着到心口那种撕裂般的痛楚愈发剧烈，仿佛有什么东西正在从他的身体里生生抽离。

“林妹妹……我仿佛听见林妹妹在叫我。”宝玉喃喃道，随后不顾一切地推开门，连鞋也顾不得穿，赤着脚便冲了出去。

袭人、麝月等人在后面惊叫着追赶，却哪里拦得住一个已经疯了心的人。宝玉踉踉跄跄地跑向潇湘馆，一路上撞翻了多少洒扫的婆子，扯破了多少红色的绸带。那原本喜庆的红，此刻在他眼中竟如鲜血般刺眼。

当他踏入潇湘馆的那一刻，漫天的冷意让他忍不住打了一个寒战。他看见了紫鹃那双肿得像桃子一样的眼睛，看见了那原本总是点着熏香的屋子里，此刻只剩下一股子纸灰的味道，和那透骨的寒凉。

“林妹妹……”宝玉扑到床前，颤抖着手去摸那张脸。

那一瞬间，世界在他眼中彻底消散了。

这不再是那个会对他笑、会对他恼、会和他一起读《西厢》的林妹妹了。这只是一具冰冷的、没有灵魂的躯壳。宝玉只觉得脑中“嗡”的一声，仿佛有一面巨大的镜子在他面前轰然粉碎。他一直以为，只要他坚持，只要他守着那份痴心，天道总会垂怜。可现实却用这冰冷的尸首告诉他：这世间，从来就没有什么圆满。他所谓的坚持，在命运的巨轮面前，不过是蚍蜉撼树。那一刻，他感到的不仅仅是悲哀，更有一种彻骨的幻灭感——原来他所热爱的、所守护的一切，在真正的权力与天意面前，竟是如此的娇弱不堪。

“啊——！”宝玉仰天发出一声惨叫，那声音不像人声，倒像是受伤的孤狼在荒野中的绝望哀嚎。他猛地喷出一口鲜血，整个人如推金山倒玉柱般栽倒在黛玉的床前。那鲜血溅在白色的床帐上，宛如一朵朵凋零的红梅。

在昏厥前的那一刻，他仿佛看见了那个疯癫的僧道。那僧人指着他笑道：“看破了么？舍下了么？这红尘万丈，不过是一场荒冢里的幻梦罢了。你本是青埂峰下一顽石，何苦来这红尘受这遭罪？”

宝玉昏迷了三天三夜。在这三天里，荣国府仿佛陷入了一种诡异的寂静。王夫人下令，黛玉的丧事要从简、从快，决不能惊动了正在“养病”的宝玉。于是，这位曾经的大观园才女，便在一副薄薄的棺木中，被几个粗壮的婆子悄悄地抬出了那道曾经繁华似锦的大门。

宝钗作为新妇，在这三天里模样出了惊人的镇定。她一边服侍婆婆，一边照料昏迷的丈夫，甚至于于还亲自打理了潇湘馆剩下的那些遗物。她看着那些残缺的诗稿，看着那把断了弦的古琴，心中亦是五味杂陈。

可只有她自己知道，每当夜深人静，她看着宝玉在梦中凄厉地呼喊着“黛玉”的名字时，她的心是何等的荒凉。她赢了这场“金玉良缘”的名分，却输掉了自己作为女人的所有尊严与指望。她守着这名存实亡的婚姻，看着这正在消散的家族，仿佛看见了自己那被礼教活活埋葬的余生。她不能哭，不能怨，甚至于于不能模样出一丝一毫的嫉妒。她必须完美地扮演好这个“宝二奶奶”的角色，哪怕她的内心早已千疮百孔。这就是她的“道”，也是她的“狱”。

那日黄昏，天边聚起了一层沉甸甸的铅云，寒风在回廊间穿梭，发出一阵阵凄厉的尖啸。贾政站在荣禧堂的台阶上，望着远处逐渐没入黑暗的大观园，长长地叹了一口气。他手中的那串念珠早已停止了拨动，仿佛在这一刻，他也感到了某种无法抗拒的力量正在将这个曾经煊赫一时的家族拖向深渊。

“老爷，夜深了，回屋吧。”赵姨娘缩着脖子，在一旁小声地提醒着。

贾政没有回头，只是淡淡地应了一句：“林丫头这一走，这园子里最后的一点灵气，怕是也散干净了。”

赵姨娘撇了撇嘴，本想说几句刻薄话，可见贾政那副如丧考妣的神情，终究是没敢开口。她也感到了，这府里的气氛变了，不再是那种虚伪的繁华，而是一种真实得让人恐惧的颓败。

潇湘馆的竹林在风中沙沙作响，仿佛在为那位远去的灵魂奏响最后一曲挽歌。那原本总是干净整洁的石径上，如今已堆满了枯枝败叶。谁都知道，从今往后，这里将成为一处禁地，一处埋葬了所有天真与痴情的废墟。

当宝玉再次睁开眼时，他的眼神已经变得一片空洞。他不再哭，也不再闹，只是整日坐在窗前发呆。他的魂灵，似乎已经随着那把焚掉的诗稿，随着那口抬走的棺木，永远地离开了这具躯体。

众人知一段深情至此而绝，荣府自此更添冷意。那冷，不是节气的寒凉，而是人心枯萎后的死寂。在这末世的黄昏里，每一个人都成了这座大坟墓里的守灵人。因果之环已然扣紧，剩下的，不过是这残破躯壳在风中的最后战栗。
